 📖 Introduction
This project focuses on predicting whether a user will purchase a product based on social network advertisement data. The dataset contains user information such as age, estimated salary, and purchasing behavior. A machine learning classification model (Logistic Regression) is used to analyze patterns and make predictions.

---

 🎯 Objective
- To build a classification model using Logistic Regression  
- To predict whether a user will purchase a product  
- To understand the relationship between age, salary, and purchasing behavior  
- To evaluate model performance using different metrics
