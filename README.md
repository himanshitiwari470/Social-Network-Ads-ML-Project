# Social Network Ads Prediction (ML Project)

## 📌 Problem Statement
Predict whether a user will purchase a product based on age and salary using Social Network Ads dataset.

## 📊 Dataset
- Source: Kaggle / UCI (mention if known)
- Features: Age, Estimated Salary
- Target: Purchased (0/1)

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🧠 Model Used
- Logistic Regression

## 📈 Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Scaling
4. Model Training
5. Prediction

## 🎯 Model Performance

 Accuracy: 0.90

 Class-wise Performance:
 - Class 0: Precision = 0.94, Recall = 0.90, F1-score = 0.92
 - Class 1: Precision = 0.83, Recall = 0.89, F1-score = 0.86

Overall Performance:
- Macro Avg F1-score: 0.89
- Weighted Avg F1-score: 0.90
